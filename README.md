# Udemy Clone Project

This project is a front-end clone of Udemy, built using HTML and CSS. It mimics the layout and design of the popular online learning platform, providing a responsive and visually appealing interface for educational content.

## Features
- Home page with featured courses
- Course categories and listings
- Responsive design for all devices
- Clean and modern UI

## Technologies Used
- HTML5
- CSS3
- Flexbox & Grid for layout
- Responsive design principles
